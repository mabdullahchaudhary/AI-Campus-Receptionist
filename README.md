## ‚öôÔ∏è How It Works (The Workflow)
1. **User Interaction:** A user initiates a call directly from the web interface.
2. **Automation Trigger:** The website sends a webhook to our workflow automation engine, which handles the background logic and routing.
3. **AI Voice Agent:** The workflow communicates with **Vapi.ai** to initiate the AI voice agent, handling natural language processing and real-time conversation.

## üíª Getting Started

### 1. Clone the Repository
